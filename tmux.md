# tmux(1)

Kill Session

    tmux kill-session

Show key bindings: `ctrl-a ?`

## panes

Jump to last pane: `ctrl-a tab`

Switch panes: `ctrl-a x`

## copy-paste

Copy CWD to clipboard: `ctrl-a Y`

Copy current command line to clipboard: `ctrl-a y`

Copy-Paste mouse selections (`alt-mouse`) with `⌘-c ⌘-v`
